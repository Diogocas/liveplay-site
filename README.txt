LivePlay Site Oficial V1

Arquivos:
- index.html
- styles.css
- script.js

Como usar:
1. Suba esses 3 arquivos no seu site/hospedagem.
2. Troque no script.js:
   DOWNLOAD_URL = link do instalador
   DISCORD_URL = link do Discord/suporte
3. Se quiser, coloque um vídeo real na seção "Demo visual".

Também pode substituir textos, preço e recursos direto no index.html.
